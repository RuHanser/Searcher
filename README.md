Поисковый движок (Searcher)
Консольное приложение для поиска по текстовым файлам.  
Реализует инвертированный индекс, ранжирование результатов по релевантности.


Стек технологий
C++17 - язык программирования
CMake - система сборки
nlohmann/json - библиотека для работы с JSON
Google Test (GTest) - тестирование


Структура проекта
config.json - конфигурация движка (список файлов, лимит ответов)
requests.json - список поисковых запросов
answers.json - результат работы 
src - исходный код
include - заголовочные файлы
resources - текстовые документы для индексации


1. Клонируй репозиторий
git clone https://github.com/RuHanser/Searcher.git
cd Searcher

2. Создай папку для сборки
mkdir build && cd build

3. Запусти CMake

4. Собери проект

Для текстирование используется СTests
